# Personal Data Analyst

An AI-powered data analysis assistant that allows users to interact with datasets using natural language. Instead of writing complex Pandas code or data visualization scripts, users can simply ask questions about their data and receive insights, summaries, and visualizations generated by a Large Language Model (LLM).

## Overview

Personal Data Analyst bridges the gap between raw data and actionable insights. It combines the power of Large Language Models with Python data analysis libraries to create a conversational analytics experience.

Users can upload datasets, ask questions in plain English, and receive:

* Data summaries
* Statistical insights
* Trend analysis
* Data visualizations
* Natural language explanations
* Actionable recommendations

## Features

* 📊 Upload and analyze CSV datasets
* 🤖 Natural language interaction with data
* 📈 Automatic data visualization generation
* 🔍 Exploratory Data Analysis (EDA)
* 🧹 Data cleaning and preprocessing support
* 📋 Summary reports and insights
* 💬 Conversational interface powered by LLMs
* ⚡ Fast and intuitive workflow for non-technical users

## Installation

### Clone the Repository

```bash
git clone https://github.com/Moulika706/Personal-Data-Analyst.git
cd Personal-Data-Analyst
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows:

```bash
venv\Scripts\activate
```

Mac/Linux:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

## Running the Application

```bash
streamlit run app.py
```

After launching, open the local URL provided by Streamlit in your browser.
